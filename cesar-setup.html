6
Rodar o app localmente
Confirmar que tudo sobe sem erro na sua máquina
🟡 Importante

1. Inicie o servidor de desenvolvimento:

npm run dev

2. Abra no browser: http://localhost:3000

3. Teste as páginas principais:

localhost:3000Landing page
localhost:3000/sign-upCriar conta (Clerk)
localhost:3000/onboardingFluxo de perguntas completo
localhost:3000/vistosSeleção de vistos
ℹ️ Se aparecer erro de CLERK, confirme que o .env.local foi salvo corretamente na raiz do projeto (na mesma pasta do package.json).
✓ Testes
7
Testar o onboarding — 3 perfis diferentes
Validar que os resultados fazem sentido para cada perfil
🟡 Importante

Acesse localhost:3000/onboarding e teste os 3 cenários:

🧪
Perfil 1 — Brasileiro fora dos EUA querendo estudar
Fora dos EUA → Brasileira → Estudar → Graduação → deve sugerir F-1
🧪
Perfil 2 — Europeu querendo investir
Fora dos EUA → Europeu → Negócios → Investir e operar → deve sugerir E-2
🧪
Perfil 3 — Brasileiro nos EUA com visto H-1B
Nos EUA → Brasileira → Visto válido → H-1B → Green Card → Por mérito → deve sugerir EB-1 / EB-2 NIW
